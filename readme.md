# ToDoList App

## FEATURES

- Add a collection
- Add a task (linked to a collection)
- Display tasks of collection
- Delete a collection
- Delete a task

## STACK

- [Python](https://www.python.org/)
- [Django](https://www.djangoproject.com/)
- [HTMX](https://htmx.org/)
- [Bootstrap CSS](https://getbootstrap.com/)
